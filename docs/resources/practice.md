# 課後練習

選擇一個你能在 30–60 分鐘內人工完成、且結果容易判斷的工作，交給 Codex 從理解到交付。

## 練習題

1. 提供一個小型 repository 或 Markdown 文件。
2. 描述一項明確改動與三個驗收條件。
3. 要求先建立 branch，再執行修改與測試。
4. 檢查 diff、commit 摘要與 PR 說明。

## 回顧問題

- 哪些資訊一開始缺少，造成來回確認？
- 哪些驗證能最快提高信心？
- 哪些規則值得寫進 `AGENTS.md` 或整理成 skill？
