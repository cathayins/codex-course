# 素材轉簡報

這類任務的核心不是把文字塞進投影片，而是先建立受眾、故事線與視覺規則，再進行版面生成與逐頁檢查。

## 任務輸入

提供原始文件、既有模板、頁數範圍、受眾與簡報場合。若有禁止改寫的數據或用語，也要一併標示。

## 執行流程

1. 摘要素材並提出章節結構。
2. 確認每頁只承擔一個主要訊息。
3. 套用共用版型與視覺節奏。
4. 輸出後逐頁 render，檢查溢位與可讀性。

## 驗收重點

內容正確、層級清楚、字級可讀、圖表不誤導，而且輸出檔案能在目標軟體正常開啟。
