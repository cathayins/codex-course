# 課程導覽

快速上手 section 為所有學習者建立共同起點。你會從基本觀念開始，逐步完成一個能夠檢查與交付的任務。

## 這個 section 會學到什麼

- 理解 Codex 適合處理的任務類型。
- 熟悉 App 的主要操作區域。
- 提供足夠的目標、脈絡與限制。
- 驗證成果，而不是只接受表面上看起來正確的輸出。

## 建議順序

依照左側目錄由上而下閱讀。每頁底部會提供上一篇與下一篇，方便連續完成課程。

::: tip 學習原則
先完成一個小而清楚的任務，再逐步增加範圍與自動化程度。
:::
