# QuickStart Visual Prompts

本頁記錄 2026-07-17 使用內建 ImageGen 產生的 QuickStart 圖片。使用者提供的三張投影片只作為概念與版面氣氛參考；生成圖沒有重製原畫面、品牌、浮水印或產品 UI。

## `advisor-to-teammate.png`

- Use case: `productivity-visual`
- Reference: `截圖 2026-07-16 19.00.07.png`，只參考左右對照與乾淨投影片氣氛。

```text
Create an original split-scene visual that explains the difference between getting advice and delegating work inside a real computer workspace. On the left, a consultant discusses one printed page and an idea sketch. On the right, a trusted teammate works inside the project environment, organizing CSV and spreadsheet files into folders and producing a finished report. Use a bright white editorial background, soft blue and violet gradients, polished realistic 3D illustration, wide 16:9 composition, and generous negative space. No logos, product UI, readable text, watermark, futuristic robot, excessive screens, or stock-photo look.
```

## `engineering-evolution.png`

- Use case: `productivity-visual`
- Reference: `截圖 2026-07-16 18.59.39.png`，只參考三階段演進概念。

```text
Create an original three-stage visual showing how software work has evolved: small inline code suggestions; a developer and AI collaborating on one problem; a person delegating a complete multi-step task to an agent that can inspect a project, edit files, run tests, and return a reviewable result. Use three equal rounded panels across a white editorial canvas, refined 3D illustration, pale blue and periwinkle accents, and a clear forward progression. No readable text, captions, logos, exact software UI, watermark, robots, or holographic clutter.
```

## `one-workspace.png`

- Use case: `productivity-visual`
- References: `截圖 2026-07-16 19.00.07.png` 與 `截圖 2026-07-16 19.01.27.png`，只參考多螢幕到單一工作台與完整生命週期概念。

```text
Create an original before-and-after visual about simplifying an engineering workflow. On the left, one person is overwhelmed by several monitors, terminals, dashboards, documents, and notifications. On the right, the same person works calmly from one coherent desktop command center that organizes planning, files, code changes, tests, review notes, documentation, and deployment status. Use a modern real-world office, a clear left-to-right transformation, polished realistic 3D illustration, cool blue and violet accents, and large shapes for classroom projection. No readable text, logos, exact product interface, watermark, robot, sci-fi holograms, or exaggerated chaos.
```
